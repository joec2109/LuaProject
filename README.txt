This game uses the LOVE2D engine. Download it at https://love2d.org/
Make sure the path to the LOVE2D engine executable file is correct on the IDE

Use of external libraries has been made in this project - each library file are contained within the 'Libraries' folder.

Also, fonts have been used, these are inside the 'Fonts' folder.

For example, for the 'HUMP' library, it's new directory will be 'C:\Program Files\LOVE\lua\hump'
